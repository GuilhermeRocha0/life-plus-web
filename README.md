O Life+ é um sistema desenvolvido para facilitar o armazenamento e o acompanhamento da saúde do usuário.
A aplicação permite guardar exames médicos (como raio-X, hemogramas, ressonâncias etc.), organizar informações importantes e controlar horários de medicamentos, 
lembrando o usuário de quando iniciou e quando precisa tomar a próxima dose.

Como executar:
git clone https://github.com/GuilhermeRocha0/life-plus-web
cd life-plus-web
npm install
npm run dev
